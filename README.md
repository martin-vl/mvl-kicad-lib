# mvl-kicad-lib
Personal kicad library


## Setup KiCAD

### Configure paths
Preferences -> Configure Paths...
```
MVL_KICAD_LIB   /Users/martin/Git/mvl-kicad-lib
```

### Add symbol libs:
Preferences -> Manage Symbol Libraries...

Add symbol libs by opening them.


### Add footprint libs:
Preferences -> Manage Footprint Libraries...

Add footprint libs by opening them.
